1. 설치 방법

 1-1) 필수 프로그램 
	가) Visual Studio Code
	나) Nodejs		 
 1-2) vsCode 실행 -> 아무 폴더 열기 -> 터미널에 다음 코드 입력하여 리액트 설치 'npm react-create-app .'
 1-3) 설치한 폴더에 첨부한 itda.zip 압축파일을 풀기
 1-4) 터미널에 'npm install'을 입력하여 필요한 필요한 npm들을 설치
 1-5) 터미널에 'npm start' 를 입력하여 실행
